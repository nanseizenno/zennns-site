# TPCA / CAE-SDB

## 状态迁移前置控制架构

全野南政｜Nansei Zenno

本网站用于公开整理 TPCA / CAE-SDB 状态迁移前置控制架构的基本思想、工程案例、技术札记和合作说明。

TPCA / CAE-SDB 面向制造系统、自动化执行单元、群控系统与数字调用系统，关注系统进入目标状态、目标执行路径或目标物理执行阶段之前的前置判定与多路径控制。

系统进入目标状态之前，现场真正需要判断的是：

为什么不能进入；
问题来自条件、许可，还是执行链；
问题属于结构缺失、动态异常，还是控制边界；
下一步应等待、重识别、回流、协调、降级、禁止、安全锁定，还是人工确认。

本体系不是普通报警诊断，不是单纯 ready 判断，也不是事后原因分析。

TPCA / CAE-SDB 的作用，是将目标状态进入前发生的 false / NG / not ready / pending，展开为可判定、可控制、可记录、可复盘的工程结构。
